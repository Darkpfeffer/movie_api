Documentation page

Welcome to my movie-API documentation! The main functionalities of the API (use 'ctrl+F' to search for them):
'CRATE', 'READ', 'UPDATE', 'DELETE'

CREATE

Add a user

Description: This will add a new user to the user's array.
Endpoint URL: '/users'
Method type: 'POST'
Format of data expected in the body of the request: 'JSON'
Format of the response data: 'JSON'

Example request body:
{
"Username": "Mary",
"Password": "example456!"
"Email": "mary@example.com"
"Birtday": "2001-10-24"
}
Example response:
{ "Username": "Mary2", "Password": "example456!", "Email": "mary@example.com", "Birthday": "2001-10-24T00:00:00.000Z", "FavoriteMovies": [], "_id": "64130ea893504592c757550f", "__v": 0 }

Add a movie to the user's favorite list

Description: This will add a movie title to the user's favorite list. Endpoint URL: '/users/:id/:movieTitle'
':id' must be replaced with the ID of the user.
':movieTitle' must be replaced with the title of the movie

Method type: 'POST'
Format of the response data: Text
Example request URL endpoint:
'/users/1/Hitman'

Example response:
Hitman has been added to user 1's favorite list

Get data about a genre by name

Description: This will return the name a description of the specific genre
Endpoint URL: '/movies/genre/:genreName'
':genreName' must be replaced with the name of the genre.

Method type: 'GET'
Format of the response data: 'JSON'

Example request URL endpoint: '/movies/genre/action'

Example response:
{ "name": "action", "description": "Action films are built around a core set of characteristics: spectacular physical action; a narrative emphasis on fights, chases, and explosions; and a combination of state-of-the-art special effects and stunt-work." }

UPDATE

Change username

Description: This will allow the user to change their username.
Endpoint URL: '/users/:id'
':id' must be replaced with the ID of the user

Method type: 'PUT'
Format of data expected in the body of the request: 'JSON'
Format of the response data: 'JSON'

Example request URL endpoint:'/users/1'

Example request body:
{ "name":"Kim updated name" }

Example response:
{ "id": 1, "name": "Kim updated name", "favoriteMovies": [ "Hitman" ] }

DELETE

Remove movie from user's favorite list

Description: This will allow the user to remove a movie title from the favorite list.
Endpoint URL: '/users/:id/:movieTitle'
':id' must be replaced with the user's ID.
':movieTitle' must be replaced with the title of the movie.

Method type: 'DELETE'
Format of the response data: Text

Example request URL endpoint: '/users/2/Inception'

Example response:
Inception has been removed from user 2's array

Deregister from the site

Endpoint URL: '/users/:id'

Method type: 'DELETE'
Format of the response data: Text

Example request URL endpoint: '/users/1'

Example response:
user 1 has been deleted
