Documentation page

Welcome to my movie-API documentation! The main functionalities of the API (use 'ctrl+F' to search for them):
'CRATE', 'READ', 'UPDATE', 'DELETE'

CREATE

Add a user

Description: This will add a new user to the users array.
Endpoint URL: '/users'
Method type: 'POST'
Format of data expected in the body of the request: 'JSON'
Format of the response data: 'JSON'

Example request body:
{ "name": "Mary", "favoriteMovies": [] }

Example response:
{ "name": "Mary", "favoriteMovies": [], "id": "1ee0a9c2-b399-4a6f-ba0a-3d4f2641e989" }

Add a movie to the users favorite list

Description: This will add a movie title to the users favorite list. Endpoint URL: '/users/:id/:movieTitle'
':id' must be replaced with the ID of the user.
':movieTitle' must be replaced with the title of the movie

Method type: 'POST'
Format of the response data: Text
Example request URL:
'/users/1/Hitman'

Example response:
Hitman has been added to user 1's favorite list

READ

Main page

Description: This will take the user to the main page
Endpoint URL: '/'
Method type: 'GET'
Format of the response data: Text

Example response:
Welcome to my movies API!
